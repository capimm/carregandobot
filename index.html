FAQ - Perguntas Frequentes

O bot é gratuito?

Sim, oferecemos um plano gratuito com funcionalidades básicas. Para recursos avançados, temos planos premium.

Como configurar o bot?

Após adicionar o bot, use o comando !config ou acesse o painel web para configurar as opções.

O bot funciona 24/7?

Sim, nosso bot tem 99.9% de uptime e está sempre disponível para ajudar sua comunidade.

Preciso de permissões especiais?

Para adicionar o bot, você precisa ter permissões de administrador no servidor Discord.

Como entro em contato com o suporte?

Use nosso servidor de suporte no Discord ou envie um email para support@discordbot.com.

Posso personalizar o bot?

Sim, oferecemos várias opções de personalização através do painel de controle.
